# Локальные шрифты (не из Google Fonts)

Положите файлы с такими именами:

| Файл | Назначение |
|------|------------|
| `Adderley-Bold.woff2` | семья Adderley, начертание Bold из макета |
| `Involve-Bold.woff2` | семья Involve, начертание Bold из макета |

Поддерживаются только форматы, указанные в `src/global/styles/fonts-local.scss` (сейчас — `woff2`). Если у вас только `.woff`, добавьте вторую строку `src:` в соответствующий `@font-face`.

До появления файлов браузер запросит их как 404 и возьмёт запасной `sans-serif` из переменных — сборка Next при этом не ломается.
